MGC.Slider3d es un carousel orientado a la ejecución de presentaciones que admite tanto imágenes como vídeos en su contenido.

Desde JavaScript podemos indicarle de formas diferentes en que elemento queremos posicionarnos. El plugin de este ejemplo contiene 9 imágenes y un video.

MOVIENDO EL CARRUSEL

La instalación del plugin requiere lo siguiente:

Copiar los archivos de las licencias junto con los ficheros js en vuestro sitio web:
ej:

js/LICENSE JQuery.txt
js/LICENSE JQuery-ui.txt


Copiar y/o enlazar los siguientes archivos en vuestro sitio web respetando el orden siguiente:
ej:

js/jquery-3.2.1.min.js
js/jquery-ui.min.1.12.1.js
js/mgc.slider3d.js


...HTML
<script src="js/jquery-3.2.1.min.js" type="text/javascript">
<script src="js/jquery-ui.min.1.12.1.js" type="text/javascript">
<script src="js/mgc.slider3d.js" type="text/javascript">


Inicializar el plugin una vez leido el DOM:
ej:

...JS
$(function () {
      $('#carousel').slider3d({
             items: items,
             ...
      });
      ...
});

Las funciones javascript utilizadas para mover el carrusel son las siguientes:

{"move", n [, t1]}

Mover a derecha e izquierda. "n" indica el número de items a desplazar y su dirección (según sea positivo o negativo). "t1" es el tiempo en milisegundos que empleará en el desplazamiento (def: 600):

$('#carousel').slider3d("move", 1, 600);
$('#carousel').slider3d("move", -1);


{"moveTo", idx [, t1]}

Mover a un item determinado. "idx" indica el número de item y "t1" el tiempo de desplazamiento (def: 600):

$('#carousel').slider3d("moveTo", n, 600);
$('#carousel').slider3d("moveTo", n);


{"showControls" [, t1 | null] [, color1 | null] [, color2 | null] [, size | null] [, height | null] [, pos]}

Mostrar y ocultar controles para el desplazamiento a derecha e izquierda. Los parámetros son el tiempo de desplazamiento (def: 600), color (def: rgba(255, 255, 255, .5), color resaltado (def: rgb(255, 255, 255), tamaño (def: 1em), altura (def: 5%) y posición (def: right-left 5%):

$('#carousel').slider3d("showControls", 1000, null, null, "3em", null, "-10%");
$('#carousel').slider3d("showControls");
$('#carousel').slider3d("hideControls");


{"showIndicators" [, t1 | null] [, color1 | null] [, color2 | null] [, size | null] [, height]}

Mostrar y ocultar indicadores/selectores de posición:/

$('#carousel').slider3d("showIndicators", 1000, "lightgray", "blue", null, "-10%");
$('#carousel').slider3d("hideIndicators");


{"random" [, t1]}

Posicionamiento aleatorio. Similar a moveTo pero sustituyendo n por un número calculado internamente al azar

$('#carousel').slider3d("random", 1000);

Las funciones javascript utilizadas para animar los items del carrusel son las siguientes:

{"scaleElement" ele, [, t1 | null] [, t2 | null] [, dim]}

Desplazar el ítem seleccionado hasta otro elemento del DOM. \"ele\" es el elemento donde se posicionará el ítem, \"t1\" el tiempo de desplazamiento (def: 600), \"t2\" el tiempo que permanecerá en la nueva ubicación (>=0 def: 3000) y \"dim\" la dimensión a la que se ajustará el tamaño del elemento ("both", "width" o "Height", por defecto "BOTH").


Si el tiempo de permanencia en la nueva ubicación es cero, se tendrá que llamar a "scaleElementTerminate" para que el ítem regrese al carrusel:

$('#carousel').slider3d("scaleElement", $("#div_A"), 1000, 4000, "Both");
$('#carousel').slider3d("scaleElementTerminate");


{"scaleParent" ele, [, t1 | null] [, t2 | null] [, dim | null] [, overflow]}

Desplazar el ítem seleccionado hasta el elemento padre del carrusel. El primer parámetro es el tiempo que empleará en el desplazamiento (>0), el segundo es el tiempo que permanecerá en la nueva ubicación (>=0), el tercero la dimensión a la que se ajustará el tamaño del elemento y el cuarto el valor de la propiedad overflow del elemento padre.

$('#carousel').slider3d("scaleParent", 1000, 4000, "Both", "Visible");
$('#carousel').slider3d("scaleParentTerminate");


{"scaleParent" ele, [, t1 | null] [, t2 | null] [, overflow]}

Escalar el ítem seleccionado hasta el ancho del carrusel. El primer parámetro es el tiempo que empleará en el escalado (>0), el segundo es el tiempo que permanecerá con el nuevo tamaño (>=0) y el tercero el valor de la propiedad overflow del carrusel.

$('#carousel').slider3d("scaleWidth", 1000, 4000, "Visible");
$('#carousel').slider3d("scaleWidthTerminate");
